# Site Vision: Monroe Resource Hub
A premium, community-driven resource hub for Monroe residents. The site acts as a state-of-the-art concierge for finding food, housing, healthcare, education, and career opportunities.

## 1. Project Information
- **Stitch Project ID:** 2881024132479554062
- **Tech Stack:** Next.js, Tailwind CSS, Shadcn UI, Framer Motion.
- **Design System:** Documentation in `DESIGN.md`.

## 2. Sitemap
- [x] **Home** (`app/page.tsx`): Premium landing page with hero, categories, and spotlight.
- [ ] **Resources** (`app/resources/page.tsx`): Searchable directory of community resources.
- [ ] **Career Center** (`app/career/page.tsx`): Job board and resume builder landing.
- [ ] **About Us** (`app/about/page.tsx`): Team profiles and mission statement.
