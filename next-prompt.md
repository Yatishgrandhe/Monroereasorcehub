# Redesign the Monroe Resource Hub Homepage
A premium, highly-interactive homepage that feels like a state-of-the-art community concierge.

**DESIGN SYSTEM (REQUIRED):**
- **Theme:** Premium Dark Mode (#020617)
- **Typography:** Fraunces (Headings), Manrope (Body)
- **Background:** Deep Midnight Navy (#020617)
- **Primary Accent:** Primary Indigo/Blue (#4F46E5)
- **Aesthetic:** Glassmorphism, Bento grids, generous whitespace.

**Page Structure:**
1. **Interactive Hero Section:** Headline: "Everything Monroe, in one place." Pill-shaped "Get Started" button.
2. **Glass Category Cards:** Horizontal grid (Food, Health, Education, Housing, Family).
3. **Bento Grid:** Featured resource partners with high-quality imagery.
4. **Impact Carousel:** Testimonials with soft-translucent backgrounds.
5. **Interactive Map:** Dark-themed minimalist map.
6. **Footer:** Multi-column layout with social links and newsletter.
